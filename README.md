# ntufar/homebrew-tap

A Homebrew tap for distributing macOS apps maintained by Nicolai Tufar.

## Maintainer

- **Name:** Nicolai Tufar
- **Email:** ntufar@gmail.com
- **GitHub:** https://github.com/ntufar

## Available casks

- `mcfind` — Fast macOS file search utility inspired by Everything on Windows

## Install

```bash
brew tap ntufar/tap
brew install --cask mcfind
```

You can also install directly from this repository:

```bash
brew install --cask ntufar/tap/mcfind
```

## About McFind

- **Homepage:** https://ntufar.github.io/mcfind/
- **Source downloads:** https://github.com/ntufar/mcfind/releases

## Uninstall

```bash
brew uninstall --cask mcfind
```
